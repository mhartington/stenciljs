<h4>API</h4>

|              |                            |
|--------------|----------------------------|
| `@State()`   | Internal State             |
| `@Watch()`   | Callback when prop changes |
| `@Element()` | Direct DOM node of element |
| `@Method()`  | Exposed public method      |
| `@Event()`   | Custom Event declaration   |
| `@Listen()`  | listens for DOM events     |
